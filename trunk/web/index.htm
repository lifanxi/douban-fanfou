豆饭

简介
豆瓣是一个可以分享有关电影、书籍、音乐的相关信息的网站。饭否是一个微博网站。
豆饭就是一个用于把豆瓣与饭否功能进行整合的插件，通过使用它可以轻松的将豆瓣上的信息分享到饭否上。
系统要求
豆饭可以工作在Mozilla Firefox 2.0-3.6.*中，标准版本需要在Firefox中安装Greasemonkey插件，xpi版本可以直接安装使用。
安装步骤
目前豆饭的最新版本是：2.0　　发布于2010年12月1日。
如果希望试用开发中的尚未发布的最新的版本：参考开发人员章节。
标准版(推荐)
1. 安装Firefox浏览器
  豆饭是为Firefox浏览器设计的，必须安装Firefox浏览器才可以使用豆饭。
  Firefox下载地址：http://www.getfirefox.com
2. 安装Greasemonkey插件
  Greasemonkey插件用于执行用户自定义的脚本，对各种网站进行扩展，豆饭其实就是一个Greasemonkey的脚本。
  Greasemonkey插件安装地址：http://addons.mozilla.org/en-US/firefox/addon/748
3. 安装豆饭
  点击下面任意一个下载地址，即会弹出安装对话框。如果没有弹出安装对话框，而出现下载对话框，则可能是因为Greasemonkey插件没有正确安装，返回第二步安装并重新启动Firefox。
  脚本下载地址：本地下载   userscript.org下载

使用方法
安装豆饭后，在打开豆瓣网站的资源页面时，会出现“分享到饭否”按钮，点击后会弹出输入对话框，里面显示了当前页面上的信息和可供修改附注，输入附注后点确定即可将当前页面的信息和附注分享到饭否上。如果反悔了不想分享了，可以在输入附注的对话框中点“取消”。目前这个界面还比较简陋，以后的版本中会对此进行改进。
同时，在“豆瓣广播”的页面上，会出现“饭否”按钮，点击这个按钮就可以同时把消息发布在豆瓣和饭否上。而原来的“我说”按钮则保持原来的功能，只把消息发布到豆瓣上。那如果要只发布到饭否呢？那您还跑豆瓣来做什么？:D
注意：在打开浏览器后第一次使用这个功能时，会弹出用户名与密码对话框，这时需要输入您在饭否上的注册名和密码才可以成功分享信息。在输入过一次用户名密码后，系统不会再次提示输入直到关闭浏览器。
